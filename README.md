# 프로젝트 명: SlimeSurvival

## 프로젝트 설명
- **설명: 스테이지마다 다가오는 슬라임을 쓰러트려 살아남는 서바이벌 게임**    
- **컨셉: 물량 막기**    
- **장르: 슬라임으로부터 살아남는 서바이벌**    
- **프로젝트 소스 코드 링크: [슬라임서바이벌_소스코드](https://github.com/ErmaelStarius/-HomeWork--2D-Unity)**  
- **멤버: 이재형, 강수지, 서영현, 최도규**
